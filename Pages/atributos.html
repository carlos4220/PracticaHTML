Atributos HTML


No Elemento Definicion
1 id="texto" Establece un indicador único a cada elemento
2 class="texto" Establece la clase CSS que se aplica a los estilos del elemento
3 style="texto" Aplica de forma directa los estilos CSS de un elemento
4 title="texto" Establece el título del elemento (Mejora la accesibilidad)
5 lang="codigo" Indica el idioma del elemento
6 xml:lang="codigo" Indica el idioma del elemento, aunque tiene más prioridad que el atributo anterior y es obligatorio si se incluye el atributo lang
7 onclick Ejecuta la acción cuando se realiza un clic sobre el elemento
8 ondblclick Ejecuta la acción cuando se realiza un doble clic sobre el elemento
9 onmousedown Ejecuta la acción cuando se detecta el botón pulsado del ratón
10 onmouseup Ejecuta la acción cuando se detecta que se ha soltado el botón del ratón
11 onmousemove Ejecuta la acción cuando se detecta el movimiento del ratón sobre el elemento
12 onmouseout Ejecuta la acción cuando el ratón abandona el elemento
13 onmouseover Ejecuta la acción cuando se detecta que el ratón se sitúa sobre el elemento
14 onload Ejecuta la acción cuando se carga el documento
15 onunload Ejecuta la acción cuando se abandona el documento
16 onresize Ejecuta la acción cuando se ha modificado el tamaño de la ventana del navegador
17 onkeydown Ejecuta la acción cuando se detecta que la tecla esta pulsada
18 width Define el ancho de un elemento
19 height Define el alto de un elemento
20 onkeypress Ejecuta la acción cuando se pulsa una tecla
